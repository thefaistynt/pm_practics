# Фаза 1: Выбор концепции проекта

## Концепции

В результате обсуждения, было выбрано 4 концепции:

Веб-сервис для студентов и преподавателей, который помогает вести учебные проекты: задачи, роли, дедлайны, отчёты, вклад участников.
Приложение для отслеживания привычек, целей и прогресса с аналитикой и напоминаниями.
Система для бронирования аудиторий, переговорных, техники (проектор, камеры, ноутбуки).
Онлайн-сервис для создания опросов, голосований и принятия решений в группах (клубы, учебные группы, команды).
