# Фаза 1: Выбор концепции проекта

## Концепции

В результате обсуждения, было выбрано 4 концепции:

1. Веб-сервис для студентов и преподавателей, который помогает вести учебные проекты: задачи, роли, дедлайны, отчёты, вклад участников.
2. Приложение для отслеживания привычек, целей и прогресса с аналитикой и напоминаниями.
3. Система для бронирования аудиторий, переговорных, техники (проектор, камеры, ноутбуки).
4. Онлайн-сервис для создания опросов, голосований и принятия решений в группах (клубы, учебные группы, команды).

## Критерии

1. Сложность реализации проекта
2. Практическая применимость
3. Уровень конкуренции
4. Сложность поддержки и сопровождения
