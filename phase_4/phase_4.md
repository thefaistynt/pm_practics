# Фаза 4. Прогнозирование

## Разбиение на задачи
1. Декомпозировать проект на задачи (20-25 задач).
2. Определить состав команды с делением на специальности.
3. Определить ресурсные требования для каждой задачи.
4. Определить связи предшествования между задачами.
5. Решить задачу эвристическими методами.
