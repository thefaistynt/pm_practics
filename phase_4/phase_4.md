# Фаза 4. Прогнозирование

## Разбиение на задачи
1. Декомпозировать проект на задачи (20-25 задач).
2. Определить состав команды с делением на специальности.
3. Определить ресурсные требования для каждой задачи.
4. Определить связи предшествования между задачами.
5. Решить задачу эвристическими методами.

## Трёхточечные оценки задач
1. Декомпозировать проект на задачи (20-25 задач).
- Оптимистично: 0.1 дня
- Наиболее вероятно: 0.5 дня
- Пессимистично: 1 день
3. Определить состав команды с делением на специальности.
- Оптимистично: 0.2 дня
- Наиболее вероятно: 0.5 дня
- Пессимистично: 1 день
4. Определить ресурсные требования для каждой задачи.
- Оптимистично: 0.25 дня
- Наиболее вероятно: 0.75 дня
- Пессимистично: 1.5 день
5. Определить связи предшествования между задачами.
- Оптимистично: 0.01 дня
- Наиболее вероятно: 0.5 дня
- Пессимистично: 1 день

- Оптимистично: 0.25 дня
- Наиболее вероятно: 0.5 дня
